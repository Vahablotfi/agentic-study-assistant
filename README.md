# Agentic Study Assistant

An AI-powered learning agent using Vector RAG, Graph RAG, MCP and n8n workflows.

This project is an AI agent system built for the AI Agents and Automation exam project.

The system helps students study uploaded course material by:

- answering questions from course documents,
- retrieving relevant content using Neo4j Vector RAG,
- mapping relationships between concepts using Neo4j Graph RAG,
- generating flashcards,
- triggering study workflows through n8n,
- exposing tools through an MCP server,
- interacting with the user through a real interface.

## Case

The system is designed as a reusable study assistant for students.  
For the exam prototype, the system will be tested using course material from AI Agents and Automation.

## Planned Architecture

User interface → Agent platform → MCP server → Neo4j / n8n tools
